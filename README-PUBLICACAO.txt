PACOTE DO SITE 11 - v11-professional-workshop

Conteudo desta pasta:
- index.html  -> arquivo principal do site
- README-PUBLICACAO.txt -> este guia rapido

Como testar no outro PC:
1) Abra a pasta.
2) Clique duas vezes no arquivo index.html.
3) O site deve abrir no navegador.

Como publicar em hospedagem:
1) Envie SOMENTE o arquivo index.html para a raiz do servidor/site.
2) Se a hospedagem exigir, mantenha o nome exatamente como index.html.
3) Teste o dominio apos publicar.

Observacao importante:
- Este site usa imagens e fontes online (Unsplash e Google Fonts).
- Por isso, o PC/servidor precisa de acesso a internet para visual completo.

Opcional para envio por e-mail:
- Compacte esta pasta em .zip e envie o arquivo zipado.
